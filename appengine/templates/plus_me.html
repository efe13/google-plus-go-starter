Plus.Me

If you want to get the current user's public profile, you can write: userProfile, err := p.People.Get("me").Do()

The special value "me" is used instead of a profile ID and refers to the current user. Associating the current user with their profile by using the "me" value requires OAuth authorization from the user with the plus.me scope.
